ПРАВИЛА ЗАПОЛНЕНИЯ:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. id / slug
   — Возьми из URL страницы последний сегмент
   — Пример: /catalog/outdoor-pools/malibu-sun-club → "malibu-sun-club"

2. translations
   — ru: название, адрес, описание на русском (с сайта)
   — uz: переведи название и адрес на узбекский; описание — краткий перевод или null
   — en: переведи название и адрес на английский; описание — краткий перевод или null
   — Адрес всегда заканчивай на ", Ташкент"

3. coordinates
   — Если на странице есть карта — возьми координаты с карты
   — Если карты нет — null для обоих полей

4. phone
   — Формат: "+998 XX XXX-XX-XX"
   — Если несколько номеров — бери первый

5. telegram
   — Полная ссылка https://t.me/... если есть на странице
   — Если нет — null

6. website
   — Ссылка на официальный сайт бассейна если указана
   — Если нет — null

7. gallery
   — Пути в формате: "/images/pools/{slug}/1.jpg", "/images/pools/{slug}/2.jpg" и т.д.
   — Количество = сколько фото есть на странице (обычно 3–5)
   — Максимум 5 фотографий

8. poolLength
   — Длина бассейна в метрах (только число)
   — Пример: 50, 25, 15
   — Если не указано — null

9. poolDepthMin / poolDepthMax
   — Глубина в метрах (число с точкой)
   — Пример: 1.2, 2.5
   — Если не указано — null

10. rating
    — Число от 1.0 до 5.0
    — Если на сайте рейтинг звёздами — конвертируй (5 звёзд = 4.8, 4 звезды = 4.0)
    — Если рейтинга нет — 4.0 (по умолчанию)

11. reviewCount
    — Количество отзывов (целое число)
    — Если не указано — 0

12. featured
    — Всегда false (это решает редактор Basen.uz вручную)

13. createdAt
    — Всегда текущая дата в формате: "2026-06-29T00:00:00Z"
