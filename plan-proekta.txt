# Проект Basen.uz

Я начинаю разработку проекта **Basen.uz**.

**Basen.uz** — это современный веб-сервис-каталог всего, что связано с водой в Узбекистане.

На первом этапе проект посвящен только бассейнам Ташкента.

В будущем проект будет расширен на весь Узбекистан и будет включать:

* Открытые бассейны
* Крытые бассейны
* Спортивные бассейны
* Детские бассейны
* Бассейны в гостиницах
* Аквапарки
* SPA
* Сауны
* Хамам
* Школы плавания
* Тренеров
* Другие объекты, связанные с водой

Но сейчас мы разрабатываем **только первую версию (Frontend MVP)**.

---

# Технологии

Проект разрабатывается на:

* Nuxt 3
* Vue 3
* TypeScript
* Tailwind CSS
* Pinia
* VueUse

Backend пока отсутствует.

Все данные должны храниться в локальных JSON-файлах.

Архитектура должна быть такой, чтобы в будущем можно было безболезненно заменить JSON на API.

---

# Главная цель

Создать лучший каталог бассейнов Ташкента.

Проект должен выглядеть как современный коммерческий продукт уровня Booking, Airbnb, Google Maps и Apple.

Пользователь должен иметь возможность легко найти бассейн по любым параметрам.
