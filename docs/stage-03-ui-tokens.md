# Этап 3 — UI + Design Tokens

## 1. Философия дизайна

**Принципы:** чистота Apple, функциональность Booking, тепло Airbnb, скорость Linear.

- Контент важнее декора — никаких лишних элементов
- Каждый пиксель служит пользователю
- Mobile-first — всё начинается с экрана 390px
- Скорость восприятия — пользователь понимает страницу за 3 секунды

## 3. Типографика

**Шрифт:** `Inter` — отличная поддержка кириллицы, системный вид, читаемость на Android.

```

```

### Шкала размеров

| Токен | px | rem | Применение |
|-------|----|-----|-----------|
| `text-xs` | 12 | 0.75 | Лейблы, мета-инфо |
| `text-sm` | 14 | 0.875 | Вторичный текст, теги |
| `text-base` | 16 | 1 | Основной текст, описания |
| `text-lg` | 18 | 1.125 | Подзаголовки карточек |
| `text-xl` | 20 | 1.25 | Названия бассейнов |
| `text-2xl` | 24 | 1.5 | Заголовки секций |
| `text-3xl` | 30 | 1.875 | H2 страниц |
| `text-4xl` | 36 | 2.25 | H1 страниц |
| `text-5xl` | 48 | 3 | Hero-заголовок |

### Вес шрифта

| Токен | Применение |
|-------|-----------|

## 7. Анимации и переходы

**Принцип:** быстро, незаметно, осмысленно. Анимация не должна задерживать пользователя.

| Элемент | Анимация | Длительность |
|---------|----------|-------------|
| Кнопки (hover) | `scale-[1.02]` + `shadow-md` | 150ms ease |
| Карточки (hover) | `translateY(-4px)` + `shadow-lg` | 200ms ease |
| Страничные переходы | `opacity` + `translateY(8px)` | 250ms ease-out |
| Фильтры (открытие) | `height` slide down | 200ms ease |
| Drawer (мобайл) | slide from bottom | 300ms ease-out |
| Skeleton loader | pulse (opacity 0.5→1) | 1.5s infinite |
| Маркер карты (hover) | `scale-[1.2]` | 150ms |
| Toast уведомление | slide + fade | 300ms |

**Принцип reduced-motion:** все анимации обёрнуты в `@media (prefers-reduced-motion: reduce)`.

## 10. Dark Mode

**Решение для MVP:** Dark Mode не включаем. Причины:
- 85% аудитории — Android, большинство используют светлую тему
- Удваивает объём UI-работы
- Контент (фото бассейнов) лучше на светлом фоне

**В v2:** добавить через CSS-переменные + класс `dark` на `<html>`.
