ПЛАН МОНЕТИЗАЦИИ И РОСТА BASEN.UZ
Цель: выйти на $1500-2000/мес стабильного дохода
==================================================

ТЕКУЩЕЕ СОСТОЯНИЕ (июль 2026)
- 50 бассейнов в каталоге, растёт вручную
- GA4 + Yandex Webmaster подключены, sitemap/robots готовы
- Дохода пока нет, чистый каталог без монетизации
- Трафик — органика, пока небольшой (сайт молодой)

ГЛАВНЫЙ ПРИНЦИП: без трафика монетизация не работает.
Прежде чем продавать рекламу — нужно от 5-10k визитов/месяц.
Поэтому план в два параллельных трека: РОСТ ТРАФИКА и МОНЕТИЗАЦИЯ,
но деньги реально пойдут после того как трафик подрастёт.


==================================================
ТРЕК 1 — РОСТ ПОСЕЩАЕМОСТИ (без этого дальше ничего не работает)
==================================================

1. SEO-контент (бесплатно, самый важный канал)
   - Статьи-подборки: "Лучшие бассейны Ташкента 2026", "Бассейны для детей
     в Юнусабаде", "Где поплавать зимой в Ташкенте" и т.д. — по одной
     статье на район/категорию, это уже 15-20 готовых SEO-страниц
   - Страницы районов и категорий (уже есть) — дозаполнить уникальным
     текстом 300-500 слов на каждую, сейчас там мало контента
   - Раздел "Блог" или "Гид" — сезонные темы (открытие летнего сезона,
     аквапарки на майские праздники, топ для дня рождения ребёнка)
   - Внутренняя перелинковка между бассейнами одного района

2. Соцсети (бесплатно, но нужно время/человек)
   - Telegram-канал Basen.uz: свежие бассейны, акции, сезонные подборки,
     репост из instagram. Telegram — основной канал в Узбекистане
   - Instagram: короткие рилсы с бассейнами (видео от владельцев или
     собственная съёмка), сторис-опросы "куда сходить в выходные"
   - Раз в неделю пост-подборка "Топ-5 бассейнов района X"

3. Локальная реклама с малым бюджетом (сезонный буст)
   - Яндекс.Директ / Google Ads на пиковый сезон (май-август),
     $50-100/мес на самые горячие запросы ("бассейн Ташкент",
     "аквапарк Ташкент")
   - Таргет в Instagram/Facebook по Ташкенту, $30-50/мес тест

4. Партнёрства
   - Договориться с 2-3 крупными Telegram-каналами про Ташкент
     (афиши, скидки, семейный отдых) на взаимный пиар
   - Микро-инфлюенсеры (5-20k подписчиков) — бартер: бесплатный вход
     в бассейн в обмен на обзор/сторис со ссылкой на сайт

Метрика успеха трека 1: 8-10k уникальных визитов/мес к 3-му месяцу.


==================================================
ТРЕК 2 — МОНЕТИЗАЦИЯ (запускать по мере роста трафика)
==================================================

A) Платные премиум-размещения для владельцов бассейнов (главный источник)
   - "Топ в категории/районе" — бассейн поднимается выше в списке
     и попадает в блок Featured на главной
   - Цена: $20-40/мес за объект (для Ташкента это адекватно для
     заведения с баром/рестораном при бассейне)
   - Что даёт: больше фото в галерее, бейдж "Рекомендуем",
     приоритет в поиске и на карте
   - Реалистично: 25-35 премиум-объектов x $30 = $750-1050/мес

B) Баннерная / нативная реклама смежного бизнеса
   - Не Google AdSense (копейки при таком трафике), а прямая продажа
     баннеров по договорённости: доставка еды, такси, купальники/
     плавательная экипировка, детские лагеря, фитнес-клубы
   - Баннер на главной/в каталоге: $50-150/мес за партнёра
   - 2-4 партнёра = $150-500/мес

C) Лидогенерация / комиссия за бронирование
   - Кнопка "Забронировать место/шезлонг" через сайт → заявка
     уходит владельцу в Telegram, с бассейна берётся комиссия
     10-15% с брони или фикс $1-2 за подтверждённую заявку
   - Работает только для топовых летних бассейнов с баром (Malibu,
     Radisson, Wyndham и т.п.) — там уже есть спрос
   - Оценка: $150-400/мес при небольшом объёме броней

D) Платный "кабинет владельца" (когда объектов станет 100+)
   - Владелец сам обновляет цены/фото/расписание за подписку
     $10-15/мес — экономит тебе время на ручное обновление
   - Пока рано технически (нет бэкенда/авторизации), закладывать
     на этап после 100+ объектов и появления реального спроса

ИТОГО реалистичный микс при трафике 8-10k/мес:
  A) $750-1050 + B) $150-500 + C) $150-400 = $1050-1950/мес
  → цель $1500-2000 достижима за счёт A+B+C без захода в разработку
    сложного бэкенда с оплатами


==================================================
ТРЕК 3 — РАСШИРЕНИЕ ПРОЕКТА (когда база устоялась)
==================================================

1. Больше объектов — 100-150 бассейнов по Ташкенту, затем
   Самарканд/Бухара/Фергана (это кратно увеличивает SEO-охват:
   "бассейны Самарканда" — отдельный низкоконкурентный кластер)

2. Смежные категории на той же платформе: фитнес-клубы с бассейном,
   spa/бани, детские лагеря с бассейном — тот же каталог, шире рынок
   рекламодателей для трека 2

3. Онлайн-оплата брони через Payme/Click — превращает лидген (C)
   в полноценную комиссию с реальных платежей, поднимает средний чек

4. Простая CRM/админка для себя (не для клиентов) — чтобы быстрее
   добавлять бассейны и вести список рекламодателей/оплат вместо
   ручного редактирования JSON


==================================================
ПОРЯДОК ДЕЙСТВИЙ ПО МЕСЯЦАМ
==================================================

Месяц 1-2: контент (статьи/районы) + Telegram-канал + Instagram,
           параллельно продолжать добавлять бассейны (цель 80-100 шт)
Месяц 2-3: первый тестовый рекламный бюджет ($50-100), выйти на
           первых 5-10 премиум-клиентов среди владельцев бассейнов
           с летним баром (у них есть бюджет и мотивация)
Месяц 3-4: подключить баннеры смежного бизнеса, запустить
           лидогенерацию для топ-объектов
Месяц 4-6: при трафике 8-10k/мес — довести премиум-размещения
           до 25-35 объектов, это и даёт целевые $1500-2000/мес

Ключевой риск: без трафика никто не заплатит за рекламу.
Поэтому первые 2 месяца — упор именно на посещаемость, а не продажи.
