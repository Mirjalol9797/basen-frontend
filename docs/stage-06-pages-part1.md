# Этап 6 — Страницы (часть 1: Главная, Каталог, Детальная)

---

## 1. Главная страница `/`

**Назначение:** первое впечатление, быстрый вход в каталог, демонстрация ценности продукта.

### Блоки (сверху вниз)

**Hero**
- Фоновое фото/видео бассейна, затемнение через gradient overlay
- H1: "Найди лучший бассейн в Ташкенте"
- Подзаголовок: "50+ бассейнов. Цены, расписание и фото без звонков."
- Строка поиска (крупная, по центру) с placeholder "Поиск по названию или району..."
- Быстрые теги под строкой: "Открытые", "Детские", "Спортивные", "Рядом со мной"

**Категории**
- 6 плиток в сетке (2 колонки мобайл / 3 десктоп)
- Каждая: фото-фон + иконка + название + счётчик объектов
- Клик → `/category/[type]`

**Сезонный блок** *(динамический)*
- Летом (май–сентябрь): "Открытые бассейны — работают сейчас" → карточки открытых
- Зимой (октябрь–апрель): "Крытые бассейны — тепло в любую погоду" → карточки крытых
- Логика: `const month = new Date().getMonth()`, нет бэкенда — всё на JS

**Популярные бассейны**
- Заголовок "Популярные в Ташкенте"
- 6–8 карточек `PoolCard` (горизонтальный scroll на мобайл, grid на десктоп)
- Источник: `featured: true` в pools.json

**Карта-превью**
- Статичная миниатюра карты (скриншот или лёгкий MapLibre)
- Счётчик: "48 бассейнов на карте"
- Кнопка: "Смотреть на карте →"

**Статистика**
- 3 цифры в ряд: "50+ бассейнов", "6 типов", "Все районы Ташкента"

**Footer** — общий для всего сайта

### UX
- Строка поиска в фокусе при загрузке (autofocus на десктоп, не на мобайл)
- Быстрые теги под поиском — самые частые запросы по анализу аудитории
- Сезонный блок создаёт ощущение актуальности без бэкенда
- Hero-CTA не "зарегистрируйся", а "найди бассейн" — снижает барьер

### SEO
```
title: "Бассейны Ташкента — каталог, цены, расписание | Basen.uz"
description: "Все бассейны Ташкента в одном месте. Открытые, крытые, детские, спортивные. Цены, расписание, фото и адреса."
og:image: hero-фото главной
canonical: https://basen.uz/
schema: WebSite + SearchAction
```

## 2. Каталог `/catalog`

**Назначение:** основная рабочая страница — просмотр и фильтрация всех бассейнов.

### Блоки

**Шапка каталога**
- H1: "Бассейны Ташкента"
- Счётчик результатов: "Найдено 48 бассейнов"
- Переключатель вид: `[Список] [Карта]` — при карте рендерится `MapView`
- Кнопка "Фильтры" (мобайл) → открывает drawer снизу

**Фильтры (десктоп — боковая панель, мобайл — bottom drawer)**

| Фильтр | Компонент | Поведение |
|--------|-----------|-----------|
| Тип бассейна | Чекбокс-группа | Мульти-выбор |
| Район | Select dropdown | Один выбор |
| Цена (разовое) | Range slider | 0–200 000 сум |
| Услуги | Чекбокс-группа | Мульти-выбор |
| Сезон | Toggle (лето / круглый год) | Один выбор |
| Сортировка | Dropdown | Рейтинг / Цена ↑↓ / Новые |

- Кнопка "Сбросить фильтры" (появляется при активных фильтрах)
- Применение фильтров: мгновенно (без кнопки "Применить"), computed в store

**Сетка карточек**
- Компонент `PoolCard` для каждого бассейна
- Skeleton-loader при инициализации (имитация загрузки для UX)
- Пустое состояние: иллюстрация + "Ничего не найдено. Попробуйте изменить фильтры."
- Пагинация: 12 карточек на страницу, кнопка "Показать ещё" (infinite scroll v2)

### UX
- URL отражает активные фильтры: `/catalog?type=open&district=yunusabad` → shareable ссылка
- При открытии Карты — фильтры остаются активными, маркеры фильтруются тоже
- На мобайл фильтры-теги показываются горизонтальным скролом над сеткой

### SEO
```
title: "Каталог бассейнов Ташкента — все 50+ объектов | Basen.uz"
description: "Полный каталог бассейнов Ташкента. Фильтрация по типу, цене, району. Открытые, крытые, детские, спортивные."
canonical: https://basen.uz/catalog
schema: ItemList (список бассейнов)
```

## 3. Детальная страница `/catalog/[slug]`

**Назначение:** вся информация о конкретном бассейне — принятие решения и контакт.

### Блоки

**Breadcrumbs**
`Главная → Каталог → Детские бассейны → Malibu Sun Club`

**Галерея фото**
- Hero-галерея: большое фото + 4 миниатюры (десктоп grid)
- На мобайл: горизонтальный swipe-слайдер с точками-индикаторами
- Кнопка "Все фото (8)" → модалка с полной галереей

**Заголовок и бейджи**
- H1: название бассейна
- Бейджи: тип категории, сезон, рейтинг ★4.6, "Открыто сейчас" / "Закрыто"

**Двухколоночный layout (десктоп) / Линейный (мобайл)**

*Левая колонка — контент:*
- Описание (2–4 абзаца)
- Услуги (иконки + название)
- Расписание (таблица по дням, текущий день выделен)
- Цены (таблица: тип → сумма)
- Встроенная карта (MapLibre, маркер бассейна, кнопка "Маршрут")
- Отзывы (список, звёзды, дата, имя автора)

*Правая колонка — сайдбар (sticky на десктоп):*
- Цены (краткая версия)
- Часы работы сегодня
- Адрес
- Кнопка "Написать в Telegram" (если есть telegram)
- Кнопка "Позвонить" (tel: ссылка)
- Рейтинг + количество отзывов

**Мобайл — sticky footer:**
```
[★ 4.6  Malibu Sun Club]  [Telegram] [Позвонить]
```
Всегда виден при скролле, не перекрывает контент.

**Блок "Похожие бассейны"**
- 4 карточки той же категории
- Горизонтальный скролл на мобайл

### UX
- "Открыто сейчас" вычисляется на клиенте через `isOpenNow(schedule)` — без бэкенда
- Галерея открывается в модалке с keyboard navigation (← →, Escape)
- Sticky сайдбар исчезает при достижении footer
- При отсутствии Telegram — показывается только "Позвонить"

### SEO
```
title: "Malibu Sun Club — цены, расписание, фото | Basen.uz"
description: "Malibu Sun Club в Юнусабаде. Открытый бассейн. Цены от 50 000 сум. Телефон, адрес, отзывы."
og:image: gallery[0]
canonical: https://basen.uz/catalog/malibu-sun-club
schema: LocalBusiness + AggregateRating + OpeningHoursSpecification
```
