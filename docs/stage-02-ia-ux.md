# Этап 2 — Информационная архитектура + UX

## 1. Структура сайта (страницы)

```
basen.uz/
├── /                          — Главная
├── /catalog                   — Каталог (все бассейны)
├── /catalog/[slug]            — Детальная страница бассейна
├── /map                       — Карта бассейнов
├── /category/[type]           — Категория (открытые, детские и т.д.)
├── /search                    — Результаты поиска
├── /about                     — О проекте
├── /faq                       — Частые вопросы
└── /404                       — Страница ошибки
```

### Категории (из анализа рынка, 6 типов):
```
/category/open         — Открытые бассейны
/category/indoor       — Крытые бассейны
/category/children     — Детские бассейны
/category/sport        — Спортивные бассейны
/category/hotel        — Бассейны в гостиницах
/category/aquapark     — Аквапарки
```

## 2. Навигация

### Десктоп (header):
```
Лого | Каталог ▾ | Карта | О проекте | [Поиск 🔍]
         └── Открытые
         └── Крытые
         └── Детские
         └── Спортивные
         └── В гостиницах
         └── Аквапарки
```

### Мобильный (bottom nav — Android-паттерн):
```
[🏠 Главная] [📋 Каталог] [🗺 Карта] [🔍 Поиск]
```

**Решение:** нижняя навигация на мобильном — стандарт Android-приложений. Пользователи привыкли, не нужно думать. Hamburger-меню убираем — он замедляет поиск.

### Breadcrumbs (для SEO и ориентации):
```
Главная → Каталог → Детские бассейны → Aqva Sport
```

## 3. Связи между страницами

```
Главная
  ├─→ Каталог (кнопка "Все бассейны")
  ├─→ Карта (кнопка "Смотреть на карте")
  ├─→ Категория (6 плиток категорий)
  └─→ Детальная страница (популярные бассейны)

Каталог
  ├─→ Детальная страница (клик по карточке)
  ├─→ Карта (переключатель вид: список / карта)
  └─→ Категория (фильтр по типу)

Карта
  ├─→ Детальная страница (клик по маркеру → попап → кнопка)
  └─→ Каталог (кнопка "Список")

Детальная страница
  ├─→ Каталог (хлебные крошки, кнопка "Назад")
  ├─→ Категория (тег типа бассейна)
  ├─→ Карта (маленькая встроенная карта)
  └─→ Другие карточки (блок "Похожие бассейны")
```

## 4. Пользовательские пути (User Flows)

### Flow 1 — Быстрый поиск (самый частый)
```
Главная → строка поиска → вводит "детский" → 
результаты → карточка → детальная страница → 
нажимает "Написать в Telegram" или "Позвонить"
```
Время: < 60 секунд

### Flow 2 — Поиск через категорию
```
Главная → плитка "Открытые бассейны" → 
каталог с фильтром → сортирует по рейтингу → 
карточка → детальная страница → контакт
```

### Flow 3 — Поиск через карту (геолокация)
```
Главная → "Смотреть на карте" → карта → 
разрешает геолокацию → видит ближайшие → 
клик на маркер → попап с фото и ценой → 
"Подробнее" → детальная страница
```

### Flow 4 — Сравнение вариантов
```
Каталог → просматривает карточки → 
открывает несколько в новых вкладках → 
сравнивает цены/расписание → выбирает → контакт
```

### Flow 5 — Мобильный пользователь (Android)
```
Google поиск "бассейн рядом" → 
попадает на SEO-страницу категории → 
видит список → фильтр по району → 
карточка → звонок одним тапом
```

## 9. URL-структура (SEO-дружественная)

| Страница | URL |
|----------|-----|
| Главная | `/` |
| Каталог | `/catalog` |
| Карта | `/map` |
| Детальная | `/catalog/aqua-sport-yunusabad` |
| Категория | `/category/children` |
| Поиск | `/search?q=детский&district=yunusabad` |
| FAQ | `/faq` |
| О проекте | `/about` |

**Правило slug:** только латиница, транслитерация, дефисы. Пример: `malibu-sun-club`, `monaco-aquapark`.
