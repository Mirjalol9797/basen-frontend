## 4. Карта `/map`

**Назначение:** визуальный поиск бассейна по расположению, геолокация.

### Блоки

**Полноэкранная карта** (MapLibre GL JS)
- Занимает весь viewport
- Маркеры всех бассейнов — кастомные иконки по типу категории (цвет из Design Tokens)
- Кластеризация при zoom out (число объектов в кластере)
- Кнопка "Найти меня" → геолокация → центрирование + радиус 3км

**Поверх карты — плавающие элементы**
- Строка поиска (сверху, полупрозрачная)
- Фильтры-теги (горизонтальный скролл): Открытые | Крытые | Детские | ...
- Кнопка "Список" → переход в `/catalog`

**Попап при клике на маркер**
```
┌─────────────────────────┐
│ [фото]   Malibu Sun Club│
│          ★ 4.6 | Открыт.│
│          от 50 000 сум  │
│     [Подробнее →]       │
└─────────────────────────┘
```
- Маленькая карточка с фото, названием, рейтингом, ценой от
- Кнопка "Подробнее" → детальная страница

**Боковая панель (десктоп)** — при клике на маркер:
- Раскрывается справа: полная карточка бассейна без перехода на новую страницу

### UX
- Карта загружается с тайлами из OpenStreetMap (бесплатно)
- Геолокация запрашивается только по клику на кнопку (не автоматически)
- На мобайл попап появляется снизу как bottom sheet
- При фильтрации — маркеры без совпадения затемняются, не скрываются

### SEO
```
title: "Бассейны Ташкента на карте | Basen.uz"
description: "Найди бассейн рядом с тобой. Интерактивная карта всех бассейнов Ташкента с адресами и маршрутами."
canonical: https://basen.uz/map
```
*Карта не prerender — SSR без JS не имеет смысла. Страница с `ssr: true`.*

## 5. Категория `/category/[type]`

**Назначение:** посадочная страница для SEO + фокусированный каталог по типу.

**Пример:** `/category/children` → "Детские бассейны Ташкента"

### Блоки

**Hero-секция категории**
- Фоновое фото типичного объекта этой категории
- H1: "Детские бассейны Ташкента"
- Подзаголовок из описания категории
- Счётчик: "17 объектов"

**Каталог** — тот же `CatalogGrid`, но предфильтрован по категории
- Боковые фильтры (без фильтра "Тип" — он уже применён)
- Та же сортировка и пагинация

**SEO-текст блок** (снизу страницы)
- 2–3 абзаца про данный тип бассейна в Ташкенте
- Для SEO-ранжирования по длинным ключевым запросам
- Написать вручную под каждую категорию

### SEO
```
title: "Детские бассейны Ташкента — цены и расписание | Basen.uz"
description: "17 детских бассейнов в Ташкенте. Цены, расписание, адреса. Безопасно для детей от 3 лет."
canonical: https://basen.uz/category/children
schema: ItemList + BreadcrumbList
```
*Все 6 категорий prerender при сборке — высокоприоритетные SEO-страницы.*

## 6. Поиск `/search`

**Назначение:** результаты поиска по строке запроса.

**URL:** `/search?q=детский+бассейн`

### Блоки

**Строка поиска** — заполнена текущим запросом, редактируемая
**Счётчик:** "По запросу «детский» найдено 12 бассейнов"
**Результаты:** те же карточки `PoolCard`
**Пустое состояние:** "Ничего не найдено по запросу «…»" + предложения категорий

### Логика поиска (клиентская, без бэкенда)

```ts
// composables/useSearch.ts
export const useSearch = (query: Ref<string>) => {
  const store = usePoolsStore()
  const q = computed(() => query.value.toLowerCase().trim())

  return computed(() =>
    q.value.length < 2 ? [] :
    store.all.filter(p =>
      p.name.toLowerCase().includes(q.value) ||
      p.address.toLowerCase().includes(q.value) ||
      p.description.toLowerCase().includes(q.value)
    )
  )
}
```

Поиск по: название, адрес, описание — мгновенно, без запросов.

### SEO
```
title: "Поиск бассейнов: «{query}» | Basen.uz"
robots: noindex  ← страницы поиска не индексируем
```

## 8. FAQ `/faq`

**Назначение:** ответы на частые вопросы + SEO по вопросительным запросам.

### Вопросы (минимум)
1. Как найти бассейн рядом с домой?
2. Цены актуальные?
3. Как добавить свой бассейн?
4. Есть ли бассейны с тренером?
5. Какие бассейны работают зимой?
6. Как связаться с бассейном через сайт?

### UX
- Аккордеон (один открыт по умолчанию)
- Якорные ссылки на каждый вопрос (`#faq-1`)

### SEO
```
title: "Частые вопросы о бассейнах Ташкента | Basen.uz"
description: "Ответы на популярные вопросы: цены, расписание, как найти бассейн рядом."
schema: FAQPage  ← даёт rich snippet в Google
canonical: https://basen.uz/faq
```
